Write-ups on [mseg-tester](https://github.com/mabels/mseg-tester), a
reboot-cycling network tester that walks a VM through every VLAN segment
and Wi-Fi network on a site, checking DHCP/DHCP6, DNS, routing, and geo
reachability on each one, and reporting the accumulated results out
through the one segment that has a route anywhere.
